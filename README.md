# Calculadora de IMC em Python

## 📌 Sobre o Projeto

Este projeto consiste em uma **calculadora de IMC (Índice de Massa Corporal)** desenvolvida em Python.
A aplicação permite calcular o IMC a partir do peso e da altura do usuário e registrar os resultados em um arquivo CSV, permitindo o armazenamento e análise dos dados posteriormente.

O objetivo do projeto é praticar conceitos fundamentais da linguagem Python, incluindo:

* Entrada e saída de dados
* Manipulação de arquivos
* Estruturas condicionais
* Organização básica de um script

## 🚀 Tecnologias Utilizadas

* Python 3
* Manipulação de arquivos CSV
* Lógica de programação

## 📂 Estrutura do Projeto

```
Calculadora-main/
│
├── BMI calculator.py   # Script principal da calculadora
├── bmi.csv             # Arquivo onde os dados são armazenados
└── README.md           # Documentação do projeto
```

## ⚙️ Como Executar o Projeto

1. Clone este repositório:

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Acesse a pasta do projeto:

```bash
cd Calculadora-main
```

3. Execute o programa:

```bash
python "BMI calculator.py"
```

## 📊 Funcionalidades

* Cálculo automático do IMC
* Classificação do resultado (baixo peso, normal, sobrepeso, etc.)
* Armazenamento dos resultados em arquivo CSV
